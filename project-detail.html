Sistem Dashboard Analitik

Proyek ini adalah sistem dashboard analitik modern yang dirancang untuk memproses dan memvisualisasikan data berskala besar secara real-time. Dengan menggunakan antarmuka mode gelap (dark mode) yang elegan, pengguna dapat memantau metrik penting perusahaan tanpa membuat mata cepat lelah.

Fitur utama dari aplikasi ini meliputi pembaruan grafik secara instan melalui WebSockets, pemilahan (filtering) data interaktif berdasarkan rentang waktu, dan pelaporan otomatis yang dapat diekspor ke berbagai format dokumen seperti PDF dan Excel. Desainnya sepenuhnya responsif, memastikan pengalaman analitik yang mulus baik di desktop maupun perangkat seluler.

Teknologi yang Digunakan:

HTML5 CSS3 (Vanilla) JavaScript (ES6+) Chart.js REST API
